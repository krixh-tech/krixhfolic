# 🕷️ KRISH Portfolio - Spider-Verse Edition 🕸️

A stunning, heavily animated portfolio website built with Material Design principles and Spider-Verse aesthetics. This portfolio showcases Krish's journey as an 11th-grade PCM student and aspiring engineer with a passion for technology.

## ✨ Features

### 🎨 Design & Animations
- **Heavy Material Design**: Comprehensive implementation of Google's Material Design system
- **Spider-Verse Theme**: Custom color scheme inspired by Spider-Man: Into the Spider-Verse
- **Animated Background**: Dynamic spider web patterns, floating particles, and geometric shapes
- **Smooth Animations**: AOS (Animate On Scroll) library with custom animations
- **Lottie Animations**: Interactive Lottie animations throughout the site
- **Parallax Effects**: Layered parallax scrolling for depth and immersion

### 🚀 Interactive Elements
- **Animated Navbar**: Material Design top app bar with scroll effects
- **Typing Animation**: Dynamic typing effect showcasing different roles
- **Animated Buttons**: Material ripple effects with custom hover animations
- **Skill Progress Bars**: Animated progress indicators with counter animations
- **Floating Icons**: Mouse-responsive floating elements around hero section
- **Custom Cursor**: Trailing effect following mouse movement

### 📱 Responsive Design
- **Mobile-First**: Fully responsive design for all screen sizes
- **Mobile Menu**: Slide-in navigation for mobile devices
- **Flexible Grid**: CSS Grid and Flexbox for optimal layouts
- **Touch-Friendly**: Optimized for touch interactions

### 🎯 Sections

1. **Hero Section**
   - Animated name with staggered letter animations
   - Dynamic typing text showcasing roles
   - Spider-Verse Lottie animation
   - Call-to-action buttons with material ripple effects

2. **About Section**
   - Personal story and journey
   - Animated statistics cards
   - Interactive hobby tags
   - Decorative animated elements

3. **Skills Section**
   - Six skill categories with progress indicators
   - Flip animations on card reveal
   - Hover effects with elevation changes
   - Percentage counters with smooth animations

4. **Projects Section**
   - Featured project showcases
   - Lottie animations for each project
   - Interactive project cards
   - Technology tags

5. **Contact Section**
   - Material Design form with validation
   - Contact information with hover effects
   - Social media links
   - Form submission with loading states

### 🛠️ Technologies Used

- **HTML5**: Semantic markup with accessibility considerations
- **CSS3**: Advanced animations, Grid, Flexbox, and custom properties
- **JavaScript**: Modern ES6+ with modular architecture
- **Material Components Web**: Google's official Material Design components
- **Lottie Animations**: Lightweight, scalable animations
- **AOS Library**: Scroll-triggered animations
- **Google Fonts**: Roboto and Orbitron font families

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation

1. **Clone or download the project files**
   ```bash
   # If using git
   git clone <repository-url>
   cd krish-portfolio
   ```

2. **Serve the files**
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   
   # Using Node.js (if you have it installed)
   npx http-server
   
   # Or simply open index.html in your browser
   ```

3. **Open in browser**
   - Navigate to `http://localhost:8000` (if using a server)
   - Or open `index.html` directly in your browser

## 🔧 Customization

### Changing Colors
Modify CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #ff1744;
    --secondary-color: #2196f3;
    --accent-color: #ffc107;
    /* ... */
}
```

### Adding New Sections
1. Add HTML structure in `index.html`
2. Style with CSS in `styles.css`
3. Add JavaScript functionality in `script.js`
4. Update navigation links

### Modifying Animations
- AOS animations: Change `data-aos` attributes
- Custom animations: Modify CSS `@keyframes`
- Lottie animations: Replace Lottie file URLs
